**HabitFlow AI**

HabitFlow AI is an intelligent productivity dashboard designed to bridge the gap between daily to-do lists and long-term habit consistency. By combining GitHub-style visual heatmaps with advanced Google Gemini AI coaching, it helps users clearly visualize progress, analyze performance, and dynamically adjust goals based on real-world execution pace.
